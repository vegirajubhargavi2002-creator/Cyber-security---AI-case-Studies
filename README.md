# Week -2
topic : T-Mobile Data Breach (2023) – Attackers exploited API vulnerabilities to steal
customer data.

In January 2023, T-Mobile disclosed a significant data breach in which attackers exploited vulnerabilities in one of its Application Programming Interfaces (APIs). The intrusion, which likely began in late November 2022, went undetected until January 5, 2023, allowing hackers to access the personal information of approximately 37 million current and former customers over several weeks. The stolen data included names, billing addresses, phone numbers, email addresses, dates of birth, account numbers, and service plan details, but did not include passwords, payment card information, or full Social Security numbers. T-Mobile quickly contained the breach and stated that the exposed information did not pose an immediate risk to customer accounts or finances. This incident was part of a series of security issues the company had faced in previous years, raising concerns about its API security practices.
